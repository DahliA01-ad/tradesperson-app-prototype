# Tradesperson App — Prototype

An early interactive prototype for an emergency tradesperson booking app.

This is a work-in-progress demo built to explore the concept and give a
feel for how the app could look and behave. It's intended for showing the
idea to potential stakeholders, not as a finished product.

## Viewing

Open `index.html` in any web browser.

## Status

Prototype — under active development. Features and design are provisional.
